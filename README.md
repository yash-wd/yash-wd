### 👋 About

From building structures to building software. After my D.C.E I spent years as a site &amp; quality engineer on large infrastructure projects — then pivoted in 2022, completed an intensive full-stack program, and have been shipping for the web ever since. Today I work as a **freelance web developer**, turning ambitious ideas into polished, production-ready products.

I care about **performance, accessibility, and clean, dependency-light code**. My portfolio is hand-built with zero runtime dependencies, scores **99–100 across Lighthouse** (Performance · Accessibility · Best-Practices · SEO), and ships behind **CI quality gates** — Lighthouse CI, pa11y (WCAG 2.1 AA) and HTML validation run on every push.
